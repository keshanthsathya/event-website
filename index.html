Rules

Read the competition rules carefully to avoid penalties.

  1. Early Buzz Rule (3-Second Challenge): Buzz early; answer within 3 seconds. Wrong answer = -2 points.
  2. Visual Silence Rule: No pointing, signaling, or non-verbal hints. First warning; repeated offense = -1 point.
  3. Mystic Swap Rule: Teams may swap ONE question with another team per round. Only the difficulty level is shown, not the question.
  4. Decoy Question Trap: One question each round is a decoy. Correct answer = Double Points; Wrong answer = -2 points. No one knows which question it is.
  5. Focus Mode Rule: Only the designated answerer may speak during questioning. Violation = question void for that team.
  6. Team Harmony Rule: Loud arguments = -1 point. STAY SHARP, STAY CLASSY.
  7. Mystic Moment Bonus: 3 correct answers in a row unlock MYSTIC MOMENT: skip one future question without penalty.
